Component Usage

Alerts are to be used when a user needs to be notified of an action they took. For example, if a user creates a new account in the system, they should be notified if the action succeed or failed. There are three styles of alerts: Success, warning and danger / failure. Depending on the action the user should receive the appropriate response.
